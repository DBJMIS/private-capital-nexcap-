# Branding assets (PCTU report)

Place the **Development Bank of Jamaica** logo here as:

`dbj-logo.svg`

The PCTU quarterly review HTML/PDF template checks for this file at `public/branding/dbj-logo.svg`. If it is missing, the report header renders as text-only (no broken layout).

Use vector SVG for best print quality. Ensure appropriate usage rights for your deployment.
